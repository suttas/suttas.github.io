I ♥ Suttas

SN12.10. Gotama

“Mendicants, before my awakening—when I was heading for awakening but not yet fully awake—I thought: ‘Oh no! People have really gotten into trouble. They are born, age, die, pass on, and are reborn again. Yet they don't see any escape from this suffering, this old age and death and so on. When will an escape from all this finally be found?’

Then I thought: ‘There will be old age and death, only if there is what? What are old age and death dependent on?’ Then, by focusing appropriately, I comprehended it, understanding that there will be old age and death, only if there is birth. Old age and death depend on birth.

Then I thought: ‘There will be birth, only if there is what? What is birth dependent on?’ Then, by focusing appropriately, I comprehended it, understanding that there will be birth, only if there is continued existence. Birth depends on continued existence.

Then I thought: ‘There will be continued existence, only if there is what? What is continued existence dependent on?’ Then, by focusing appropriately, I comprehended it, understanding that there will be continued existence, only if there is fuel/taking up. Continued existence depends on fuel/taking up.

Then I thought: ‘There will be fuel/taking up, only if there is what? What is fuel/taking up dependent on?’ Then, by focusing appropriately, I comprehended it, understanding that there will be fuel/taking up, only if there is craving. Fuel/taking up depends on craving.

Then I thought: ‘There will be craving, only if there is what? What is craving dependent on?’ Then, by focusing appropriately, I comprehended it, understanding that there will be craving, only if there are feelings. Craving depends on feelings.

Then I thought: ‘There will be feelings, only if there is what? What are feelings dependent on?’ Then, by focusing appropriately, I comprehended it, understanding that there will be feelings, only if there are sense impressions. Feelings depend on sense impressions.

Then I thought: ‘There will be sense impressions, only if there is what? What are sense impressions dependent on?’ Then, by focusing appropriately, I comprehended it, understanding that there will be sense impressions, only if there are the six senses. Sense impressions depend on the six senses.

Then I thought: ‘There will be the six senses, only if there is what? What are the six senses dependent on?’ Then, by focusing appropriately, I comprehended it, understanding that there will be the six senses, only if there are the individual’s inner aspects and body. The six senses depend on the individual’s inner aspects and body.

Then I thought: ‘There will be the individual’s inner aspects and body, only if there is what? What are the individual’s inner aspects and body dependent on?’ Then, by focusing appropriately, I comprehended it, understanding that there will be the individual’s inner aspects and body, only if there is consciousness. The individual’s inner aspects and body depend on consciousness.

Then I thought: ‘There will be consciousness, only if there is what? What is consciousness dependent on?’ Then, by focusing appropriately, I comprehended it, understanding that there will be consciousness, only if there are willed actions. Consciousness depends on willed actions.

Then I thought: ‘There will be willed actions, only if there is what? What are willed actions dependent on?’ Then, by focusing appropriately, I comprehended it, understanding that there will be willed actions, only if there is ignorance. Willed actions depend on ignorance.

So, dependent on ignorance, there are willed actions. Dependent on willed actions, [the continuation of] consciousness. Dependent on consciousness, the individual’s inner aspects and body. Dependent on the individual’s inner aspects and body, the six senses. Dependent on the six senses, sense impressions. Dependent on sense impressions, feelings. Dependent on feelings, craving. Dependent on craving, fuel/taking up. Dependent on fuel/taking up, continued existence. Dependent on continued existence, birth. And dependent on birth, there come to be old age and death, and sorrow, grief, pain, sadness, and distress. That is how this whole mass of suffering originates.

Mendicants, this is how I understood origination, gaining insight, comprehension, understanding, knowledge, and illumination of things nobody has taught before.

Then I thought: ‘There won't be old age and death, if there isn't what? Old age and death will cease, if what ceases? Then, by focusing appropriately, I comprehended it, understanding that there won't be old age and death, if there is no birth. If birth ceases, old age and death will cease.

Then I thought: ‘There won't be birth, if there isn't what? Birth will cease, if what ceases? Then, by focusing appropriately, I comprehended it, understanding that there won't be birth, if there is no continued existence. If continued existence ceases, birth will cease.

Then I thought: ‘There won't be continued existence, if there isn't what? Continued existence will cease, if what ceases? Then, by focusing appropriately, I comprehended it, understanding that there won't be continued existence, if there is no fuel/taking up. If fuel/taking up ceases, continued existence will cease.

Then I thought: ‘There won't be fuel/taking up, if there isn't what? Fuel/taking up will cease, if what ceases? Then, by focusing appropriately, I comprehended it, understanding that there won't be fuel/taking up, if there is no craving. If craving ceases, fuel/taking up will cease.

Then I thought: ‘There won't be craving, if there isn't what? Craving will cease, if what ceases? Then, by focusing appropriately, I comprehended it, understanding that there won't be craving, if there are no feelings. If feelings cease, craving will cease.

Then I thought: ‘There won't be feelings, if there isn't what? Feelings will cease, if what ceases? Then, by focusing appropriately, I comprehended it, understanding that there won't be feelings, if there are no sense impressions. If sense impressions cease, feelings will cease.

Then I thought: ‘There won't be sense impressions, if there isn't what? Sense impressions will cease, if what ceases? Then, by focusing appropriately, I comprehended it, understanding that there won't be sense impressions, if there are no six senses. If the six senses cease, sense impressions will cease.

Then I thought: ‘There won't be the six senses, if there isn't what? The six senses will cease, if what ceases? Then, by focusing appropriately, I comprehended it, understanding that there won't be the six senses, if there are no inner aspects and body. If the individual’s inner aspects and body cease, the six senses will cease.

Then I thought: ‘There won't be the individual’s inner aspects and body, if there isn't what? The individual’s inner aspects and body will cease, if what ceases? Then, by focusing appropriately, I comprehended it, understanding that there won't be the individual’s inner aspects and body, if there is no consciousness. If consciousness ceases, the individual’s inner aspects and body will cease.

Then I thought: ‘There won't be consciousness, if there isn't what? Consciousness will cease, if what ceases? Then, by focusing appropriately, I comprehended it, understanding that there won't be consciousness, if there are no willed actions. If willed actions cease, consciousness will cease.

Then I thought: ‘There won't be willed actions, if there isn't what? Willed actions will cease, if what ceases? Then, by focusing appropriately, I comprehended it, understanding that there won't be willed actions, if there is no ignorance. If ignorance ceases, willed actions will cease.

So, if ignorance totally fades away and ceases, willed actions will cease. If willed actions cease, consciousness will cease. If consciousness ceases, the individual’s inner aspects and body will cease. If the individual’s inner aspects and body cease, the six senses will cease. If the six senses cease, sense impressions will cease. If sense impressions cease, feelings will cease. If feelings cease, craving will cease. If craving ceases, fuel/taking up will cease. If fuel/taking up ceases, continued existence will cease. If continued existence ceases, birth will cease. And if birth ceases, old age and death, and sorrow, grief, pain, sadness, and distress, will cease. That is how this whole mass of suffering ceases.

Mendicants, this is how I understood cessation, gaining insight, comprehension, understanding, knowledge, and illumination of things nobody has taught before.”
